Q4) Operators What is the difference between == and ===?

== Operator

In JS == opertor used for comparing 2 values and then return answer in true and false as boolean. It is known as Equality Operator. In this operator JS will check both the values are same or not.

if we compare number value with string so JS will first convert string to number and the compare the value and it didn't check it data type.

if we compare boolean with number 1 mean (true == 1). So this operator will convert the 1 in true value and the compare both the values.

=== Operator

In JS === opertor used for comparing 2 values and their type too and then return answer in true and false as boolean. In this operator type check is important. It is known as Strict Equality Operator. In this operator JS will check both the values and types are same or not.

if we compare number value with string so JS will compare both the values and their type too. If both are same so it will return true. If any of them are different so it will return false.


Q5) How do increment (++) and decrement (--) operators work?

Increment Operator

Working of Increment Operator in JS is to add +1 in original value.
Two type Increment Operators
1) Prefix
2) Postfix.

Prefix

Prefix Operator define as ++x. It mean in prefix ++ sign place before the variable.

It will work same as its sign placement. Mean it will add +1 in original value before the variable being used in code.

Example if x = 5; and it print like console.log(++x);

it output become 6 why because before print the result Prefix add +1 in original value and then print the updated value.

Postfix

Postfix Operator define as x++. It mean in Postfix -- sign place after the variable.

It will work same as its sign placement. Mean it will add +1 in original value after the variable being used in code.

Example if x = 5; and it print like console.log(x++);

it output become 5. Because it will print the value first and then add +1 in original value but use will we will use original value which is defined at start of program.

Decrement Operator

Working of Decrement Operator in JS is to subtract -1 in original value.
Two type Decrement Operators
1) Prefix
2) Postfix.

Prefix

Prefix Operator define as --x. It mean in prefix -- sign place before the variable.

It will work same as its sign placement. Mean it will subtract -1 from original value before the variable being used in code.

Example if x = 5; and it print like console.log(--x);

it output become 4 why because before print the result Prefix subtract -1 from original value and then print the updated value.

Postfix

Postfix Operator define as x--. It mean in Postfix -- sign place after the variable.

It will work same as its sign placement. Mean it will subtract -1 in original value after the variable being used in code.

Example if x = 5; and it print like console.log(x--);

it output become 5. Because it will print the value first and then subtract -1 from original value but we will use original value which is defined at start of program.
